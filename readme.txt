== Description ==
Wordpress Live Support, a Powerful, Simple and Flexible WordPress Support. Start receiving support from our best WordPress expert through our simple Wordpress Live Support widget.  

[Read more](http://wordpress.org/plugins/wp-live-support/) about with this plugin!

<h4>Features</h4> 
* Realtime Live Support
* No special knowledge skills
* Multiple languages Support  

<h4>How Wordpress Live Support is working?</h4>
* We are using (https://www.tawk.to/) as live chat platform to offer you the best service, <strong>YOU DON'T NEED</strong> to create an account or to install anything special.

<h4>Your Privacy</h4>
* By using this plugin you agree that 'YOUR EMAIL & YOUR BLOG NAME' will be sent to our agents so we can offer you the best of our knowledge.
* If you have any questions about our Privacy, please contact us.

== Installation == 
* In your Admin, go to menu Plugins > Add
* Search for Wordpress Live Support
* Click to install
* Activate the plugin 
* Start using our Wordpress Live Support widget to connect with our best WordPress expert.
